## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage                                                            |
| ----- | ----- | ---------------------------------------------------------------- |
| xs    | 4px   | Icon gaps, inline padding, level pill padding                    |
| sm    | 8px   | Compact element spacing, avatar border width                     |
| md    | 16px  | Default card padding, card gap (D-06 target: `space-y-4`)        |
| lg    | 24px  | Section padding, card gap alternative (D-06 target: `space-y-6`) |
| xl    | 32px  | Layout horizontal padding on larger viewports                    |
| 2xl   | 48px  | OLD card gap (`space-y-12`) — removed in this phase              |
| 3xl   | 64px  | Page-level bottom padding                                        |

Exceptions:

- Avatar circle: 64px (h-16 w-16) — fixed touch-friendly size, source: CONTEXT.md D-03
- Level pill height: 24px (h-6) — compact pill, source: existing Dashboard.jsx line 656
- Practice tool circular buttons: 64px (h-16 w-16) — circular touch targets, matches current implementation
- Greeting bar: full-width strip, height driven by content (avatar 64px + vertical padding 16px = ~96px total)
- Card gap: use `space-y-4` (16px) as default; upgrade to `space-y-6` (24px) if vertical breathing feels too tight — Claude's discretion per D-06

## Color

All colors are glassmorphism overlays on the app's persistent purple gradient background: `bg-gradient-to-br from-indigo-900 via-purple-900 to-violet-900` (set by AppLayout.jsx line 58).

| Role            | Value                                                 | Usage                                                              |
| --------------- | ----------------------------------------------------- | ------------------------------------------------------------------ |
| Dominant (60%)  | indigo-900 → purple-900 → violet-900 gradient         | Full-page background (AppLayout)                                   |
| Secondary (30%) | `bg-white/10 backdrop-blur-md border border-white/20` | All glass cards including new greeting bar, dashboard cards        |
| Accent (10%)    | `indigo-400` / `indigo-300`                           | XP progress bar fill, level pill gradient highlight, active states |
| Destructive     | `red-500` / `ef4444`                                  | Destructive actions only (none in this phase)                      |

Accent reserved for:

- XP/level progress bar fills
- Level pill gradient border highlight (`#38bdf8` → `#f97316` for normal; amber gradient for prestige)
- Active indicator dot on Practice Tools (reminder button when reminder is set)
- Practice tool icon colors (amber-300 for Reminder, emerald-300 for Record, blue-300 for Recordings)

Text colors on glass surfaces:

- Primary: `text-white`
- Secondary: `text-white/70`
- Tertiary: `text-white/60`
- Accent numbers / stats: `-300` variants (e.g., `text-indigo-300`, `text-green-300`)

Skeleton / loading state: `bg-white/10 animate-pulse` (glass-system loading, NOT gray-200 from docs/DESIGN_SYSTEM.md — that applies to white-card pages, not the purple gradient dashboard)

## Component Inventory

### New: Greeting Bar

Replaces the full hero section (220-260px tall image + gradient overlay + Fireflies).

**Structure (horizontal strip):**

```
[Avatar 64px] [greeting text + "Hi, {name}!"] [Level pill]
```

**Classes:**

```
bg-white/10 backdrop-blur-md border border-white/20 rounded-xl shadow-lg
px-4 py-3 mx-auto max-w-2xl
flex items-center gap-3
```

**Avatar sub-component:**

- 48px circle (h-12 w-12) — downsized from hero's 64px to fit the strip
- `border-2 border-sky-300/50 shadow-[0_2px_12px_rgba(56,189,248,0.4)]`
- Wrapped in `<Link to="/avatars">` with accessible alt text from i18n key `avatars.title`
- Layered accessories rendered identically to current hero (reuse existing logic)
- Fallback: `bg-slate-800` circle with `🎹` emoji

**Greeting text:**

- `text-base font-semibold text-white` — "Hi, {name}!"
- i18n key: `dashboard.header.greeting`

**Level pill:**

- Reuse level pill code from Dashboard.jsx lines 654-676 exactly
- Normal: `linear-gradient(to right, #38bdf8, #f97316)` border-box
- Prestige: amber gradient border-box
- Height: h-6 (24px), rounded-full

**RTL support:** wrap in `flex-row-reverse` when `isRTL`

**Loading skeleton:**

- Single `h-14 rounded-xl bg-white/10 animate-pulse` strip (replaces hero skeleton block)
