**Pattern object shape** (per D-01 through D-09):

```javascript
{
  id: 'quarter_only_01',           // D-08: tag_prefix_NN (underscores, sequential)
  description: 'Four steady quarter notes',  // D-01: human-readable

  measureCount: 1                 // D-06: must equal beats.length
}
```

**Authoring approach for each of the 15 tags** (per D-28, research recommendation):

For each tag, write patterns in three passes to guarantee D-20 (all difficulties) and D-21 (all measure lengths):

- Pass 1: beginner patterns at 1-bar, 2-bar, 4-bar
- Pass 2: intermediate patterns at 1-bar, 2-bar, 4-bar
- Pass 3: advanced patterns at 1-bar, 2-bar, 4-bar
- Then add variety patterns to reach the minimum count per tag (8, or 12-15 for quarter-eighth)

**Tag-by-tag authoring rules:**

**Cumulative 4/4 tags (D-12, D-23):**

- `quarter-only`: Uses ONLY 'q'. Time signature 4/4. No rests (D-23). Example 1-bar: `['q','q','q','q']`. For variety use 2-bar and 4-bar with varied rhythmic groupings. Minimum 8 patterns.
- `quarter-half`: Uses 'q' and 'h'. No rests (D-23). Example: `['h','q','q']`, `['q','q','h']`. Minimum 8 patterns.
- `quarter-half-whole`: Uses 'q', 'h', 'w'. No rests (D-23). Example: `['w']` (1-bar), `['h','h']` (1-bar). Minimum 8 patterns.
- `quarter-eighth`: Uses any combination of 'q', 'h', 'w', '8'. No rests (D-23). This is the highest-demand tag. Example: `['8','8','q','h']`. Minimum 12-15 patterns.

**Rest tags (D-23 boundary — rests START here):**

- `with-quarter-rest`: Uses any prior durations + 'qr'. Must include at least one sounded note per measure (D-24). Example: `['q','qr','q','q']`. Minimum 8 patterns.
- `with-half-rest`: Uses any prior durations + 'hr'. D-24 applies. Example: `['hr','q','q']`. Minimum 8 patterns.
- `with-whole-rest`: Uses any prior durations + 'wr'. D-24 applies. IMPORTANT: since 'wr' fills an entire 4/4 measure, 1-bar patterns for this tag should use 'qr'/'hr' combinations (the tag is cumulative — child knows all rests). Use 'wr' in 2-bar and 4-bar patterns where one measure is `['wr']` and adjacent measures have sounded notes. Minimum 8 patterns.

**Dotted note tags:**

- `dotted-half`: Uses 'hd' + 'q' (hd=12 + q=4 = 16 for 4/4). Can also include prior durations. Example: `['hd','q']`. Minimum 8 patterns.
- `dotted-quarter`: Uses 'qd' + '8' as paired cell (qd=6, needs 8=2 to complete the beat grouping). Can include prior durations. Example: `['qd','8','q','q']` = 6+2+4+4=16. Minimum 8 patterns.

**Sixteenth tag:**

- `with-sixteenth`: Uses '16' plus prior durations. Example: `['16','16','8','q','h']` = 1+1+2+4+8=16. Minimum 8 patterns.

**Special context tags (D-13):**

- `compound-basic`: 6/8 time, uses ONLY 'qd'. Each measure = `['qd','qd']` (6+6=12). Minimum 8 patterns.
- `compound-mixed`: 6/8 time, uses 'qd', 'q', '8'. Example: `['q','8','qd']` = 4+2+6=12. Minimum 8 patterns.
- `syncopation-basic`: 4/4 time, off-beat emphasis using '8' and 'q'. The syncopated cell is eighth-quarter-eighth (a note starting on the "and"). Example: `['8','q','8','q','q']` = 2+4+2+4+4=16. Minimum 8 patterns.
- `syncopation-dotted`: 4/4 time, dotted syncopation using 'qd' and '8'. The syncopated cell is dotted-quarter + eighth. Example: `['qd','8','qd','8']` = 6+2+6+2=16. Minimum 8 patterns.

**3/4 exclusive tag (D-14, D-26):**

- `three-four`: 3/4 time ONLY. Uses 'q', 'h', 'hd'. NOT combined with cumulative 4/4 tags. Example: `['q','q','q']` = 12, `['hd']` = 12. Minimum 8 patterns.

**Multi-tagging (D-11, D-22):** A pattern CAN carry multiple tags where musically appropriate. For example, a pattern using 'qd' and '8' in a syncopated context could be tagged both `dotted-quarter` and `syncopation-dotted`. This reduces total unique pattern count while satisfying per-tag minimums.

**Measure sum verification:** Before finalizing each pattern, verify every measure sums to the correct sixteenth-note total:

- 4/4 = 16 units: q=4, h=8, w=16, 8=2, 16=1, qd=6, hd=12, qr=4, hr=8, wr=16
- 3/4 = 12 units
- 6/8 = 12 units

**ID naming** (D-08): Use the primary tag (hyphens replaced with underscores) + zero-padded sequential number. Examples: `quarter_only_01`, `quarter_eighth_12`, `compound_basic_03`, `three_four_05`.

**D-09:** All patterns start on beat 1. No pickup measures.

**D-10 (tied notes — Claude's discretion):** Survey the unit files during read_first. If any nodes reference tied rhythms, include tied note patterns. If none do, omit ties — they can be added in Phase 22. For VexFlow tied notes, the convention would be consecutive same-pitch durations that render with a tie arc, but since the pattern library only stores durations (not pitches), ties are implicit when the game component renders them. Omit tied notes unless a clear need is found.

**D-25:** Boss/mini-boss nodes use regular patterns at `difficulty: 'advanced'`. Do not create special boss-only patterns.

4. **Helper functions** (per D-30):

```javascript
export function getPatternsByTag(tag) {
  return RHYTHM_PATTERNS.filter((p) => p.tags.includes(tag));
}

export function getPatternById(id) {
  return RHYTHM_PATTERNS.find((p) => p.id === id) || null;
}

export function getPatternsByTagAndDifficulty(tag, difficulty) {
  return RHYTHM_PATTERNS.filter(
    (p) => p.tags.includes(tag) && p.difficulty === difficulty
  );
}
```

5. **Default export**: `export default RHYTHM_PATTERNS;`

**Music theory principles** (D-28):

- Beginner patterns: Strong beat emphasis (beats 1 and 3 in 4/4 are sounded), simple rhythmic cells repeated
- Intermediate patterns: More rhythmic variety, some weak-beat activity, longer phrases
- Advanced patterns: Syncopation elements, rhythmic density, varied beat placement
- Within each tag, patterns should progress from simple (repeated cells) to complex (varied cells)
