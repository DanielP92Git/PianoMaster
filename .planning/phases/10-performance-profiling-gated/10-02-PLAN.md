Create `public/worklets/` directory and the worklet processor file.

1. **Inline pitchy source:** Copy the PitchDetector class from `node_modules/pitchy/dist/index.js` (or the ESM bundle) directly into the top of the worklet file. This is ~5KB of code. Do NOT use `import` or `importScripts()` — inline the source for cross-browser compatibility (see RESEARCH.md pitfall on static imports in worklet scope). Only copy the PitchDetector class and its dependencies (the autocorrelation/McLeod functions). Strip any module export wrappers.

2. **Ring buffer implementation (PERF-03):**
```javascript
const ACCUMULATION_SIZE = 2048; // PERF-03: minimum samples for bass clef accuracy
const QUANTUM_SIZE = 128;       // AudioWorklet always delivers 128 frames per process() call
```

3. **PitchDetectorProcessor class:**
- Constructor: initialize `this._buffer = new Float32Array(ACCUMULATION_SIZE)`, `this._writePos = 0`, `this._detector = null`
- `process(inputs, outputs, parameters)`:
  - Guard: if no input channel data, return true
  - Copy 128-frame quantum into ring buffer using modulo write position
  - Only run detection when `writePos >= ACCUMULATION_SIZE` and `writePos % ACCUMULATION_SIZE === 0` (fire every time a full buffer of new data has arrived, ~45ms at 44.1kHz)
  - Lazy-init PitchDetector: `PitchDetector.forFloat32Array(ACCUMULATION_SIZE)` on first detection
  - Snapshot ring buffer into contiguous Float32Array (handle wrap-around via modulo read)
  - Call `findPitch(inputBuffer, sampleRate)` — `sampleRate` is available in AudioWorkletGlobalScope
  - `this.port.postMessage({ pitch, clarity })` to send result to main thread
  - Always `return true` to keep processor alive

4. Register: `registerProcessor('pitch-detector', PitchDetectorProcessor)`

**Avoid anti-patterns from RESEARCH.md:**
- Do NOT use SharedArrayBuffer (requires COOP/COEP headers that break Supabase)
- Do NOT call `importScripts()` (not available in AudioWorkletGlobalScope)
- Do NOT run detection on every 128-frame quantum (insufficient data for bass notes)
