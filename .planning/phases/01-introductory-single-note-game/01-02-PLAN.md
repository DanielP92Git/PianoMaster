## Pure Functions (testable, exported for tests)

**generateCardSequence(totalCards, targetCount, targetNote, distractorNotes):**
- Creates an array of `totalCards` card objects
- Exactly `targetCount` cards have `isTarget: true` and `pitch: targetNote`
- Remaining cards have `isTarget: false` and `pitch` randomly selected from `distractorNotes`
- Cards are shuffled (Fisher-Yates) so targets are distributed randomly
- Each card has shape: `{ id: number, pitch: string, isTarget: boolean }`
- Export this function for testing: `export function generateCardSequence(...)`

**getSpeedForCard(cardIndex):**
- Returns interval in ms based on card position (0-indexed):
  - 0-4: 2000ms (learning)
  - 5-9: 1500ms (warming up)
  - 10-14: 1200ms (challenge)
  - 15-19: 1000ms (fast)
- Export for testing: `export function getSpeedForCard(cardIndex)`

**calculateScore(correctCatches, targetCount):**
- Returns percentage: `Math.round((correctCatches / targetCount) * 100)`
- Export for testing: `export function calculateScore(correctCatches, targetCount)`

**Game loop (setTimeout-based, not setInterval):**

After each card is displayed, schedule the next card advance using `setTimeout`:
```javascript
useEffect(() => {
  if (gameState !== 'in-progress') return;
  if (currentCardIndex >= cardSequence.length) {
    // Game complete
    setGameState('complete');
    return;
  }
  tappedThisCard.current = false;

  const baseSpeed = getSpeedForCard(currentCardIndex);
  // Apply bonus speed reduction: -100ms per correct catch, floor 700ms
  const adjustedSpeed = Math.max(700, baseSpeed - (correctCatchesRef.current * 100));
  currentSpeedRef.current = adjustedSpeed;

  const id = setTimeout(() => {
    if (!isGameActiveRef.current) return;
    // Card expired — check if it was a target that was missed
    if (cardSequence[currentCardIndexRef.current]?.isTarget && !tappedThisCard.current) {
      handleMissedTarget();
    }
    setCurrentCardIndex(prev => prev + 1);
  }, adjustedSpeed);

  return () => clearTimeout(id);
}, [currentCardIndex, gameState, cardSequence]);
```

**Tap handler (D-07, D-08, D-09):**

```javascript
const handleTap = useCallback(() => {
  if (gameState !== 'in-progress') return;
  if (tappedThisCard.current) return; // prevent double-tap per card
  tappedThisCard.current = true;

  const card = cardSequence[currentCardIndexRef.current];
  if (!card) return;

  if (card.isTarget) {
    // Correct catch (D-08)
    playCorrectSound();
    setCorrectCatches(prev => prev + 1);
    correctCatchesRef.current += 1;
    setCombo(prev => prev + 1);
    comboRef.current += 1;
    setFeedbackState('correct');
  } else {
    // Wrong tap (D-09) — no lives lost, no punishment
    playWrongSound();
    setCombo(0);
    comboRef.current = 0;
    setWrongTaps(prev => prev + 1);
    setFeedbackState('wrong');
  }
  setFeedbackKey(prev => prev + 1);
  // Advance to next card immediately after tap
  setTimeout(() => setCurrentCardIndex(prev => prev + 1), 300);
}, [gameState, cardSequence, playCorrectSound, playWrongSound]);
```
