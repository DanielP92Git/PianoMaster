**Part C: Implement allowedPatterns filtering in getPattern()**

Inside `getPattern()`, when `allowedPatterns` is non-null and non-empty, skip the curated pattern path and go directly to generative. Before calling `generator.generatePattern(timeSignatureObj, difficulty)`, resolve `allowedPatterns` to numeric values and override the generation rules' `allowedSubdivisions`.

Add this logic after the `timeSignatureObj` resolution block and before the curated/generated branching:

```javascript
// When allowedPatterns is specified, skip curated patterns and constrain generation
const useAllowedPatterns = Array.isArray(allowedPatterns) && allowedPatterns.length > 0;

let result = null;

// Try curated pattern first (only when no allowedPatterns constraint)
if (!useAllowedPatterns) {
  result = await generator.getCuratedPattern(timeSignature, difficulty);
}

// Fallback to generated pattern (or forced for constrained patterns)
if (!result) {
  // If allowedPatterns specified, temporarily override the generation rules
  if (useAllowedPatterns) {
    const rules = GENERATION_RULES[difficulty] || GENERATION_RULES[DIFFICULTY_LEVELS.BEGINNER];
    const allowedValues = allowedPatterns
      .map(name => generator.getDurationValue(name))
      .filter(v => v !== null);
    if (allowedValues.length > 0) {
      const intersection = rules.allowedSubdivisions.filter(v =>
        allowedValues.some(av => Math.abs(av - v) < 0.001)
      );
      // Use intersection if non-empty; otherwise fall back to full rules
      if (intersection.length > 0) {
        rules.allowedSubdivisions = intersection;
      }
    }
  }
  result = generator.generatePattern(timeSignatureObj, difficulty);
}
```

**IMPORTANT:** The `GENERATION_RULES` object is module-level and mutating it would affect all subsequent calls. Instead of mutating `rules.allowedSubdivisions` directly, create a shallow copy:

```javascript
if (useAllowedPatterns) {
  const originalRules = GENERATION_RULES[difficulty] || GENERATION_RULES[DIFFICULTY_LEVELS.BEGINNER];
  const allowedValues = allowedPatterns
    .map(name => generator.getDurationValue(name))
    .filter(v => v !== null);
  if (allowedValues.length > 0) {
    const intersection = originalRules.allowedSubdivisions.filter(v =>
      allowedValues.some(av => Math.abs(av - v) < 0.001)
    );
    if (intersection.length > 0) {
      // Create a temporary rules copy with constrained subdivisions
      const constrainedRules = { ...originalRules, allowedSubdivisions: intersection };
      // Temporarily replace in GENERATION_RULES for the generatePattern call
      const diffKey = difficulty || DIFFICULTY_LEVELS.BEGINNER;
      GENERATION_RULES[diffKey] = constrainedRules;
      result = generator.generatePattern(timeSignatureObj, difficulty);
      // Restore original rules immediately
      GENERATION_RULES[diffKey] = originalRules;
    } else {
      result = generator.generatePattern(timeSignatureObj, difficulty);
    }
  } else {
    result = generator.generatePattern(timeSignatureObj, difficulty);
  }
}
```

Keep the rest of the function (legacy conversion, validation, fallback) unchanged.
