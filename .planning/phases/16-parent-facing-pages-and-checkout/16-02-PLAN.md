5. **Lemon.js loading:** In a `useEffect`:
   - Guard: Check if `document.querySelector('script[src*="lemon.js"]')` exists. If so, just call `setupLemon()`.
   - Otherwise: Create `<script>` tag with `src="https://app.lemonsqueezy.com/js/lemon.js"`, `defer=true`. On load: call `window.createLemonSqueezy?.()` then `setupLemon()`.
   - Append to `document.body`. Cleanup: remove script on unmount only if we added it.
   - `setupLemon()` function uses a `lemonSetupRef` (useRef, default false) to prevent double-setup in StrictMode:
     ```javascript
     function setupLemon() {
       if (lemonSetupRef.current) return;
       lemonSetupRef.current = true;
       window.LemonSqueezy.Setup({
         eventHandler: (event) => {
           if (event.event === 'Checkout.Success') {
             navigate('/subscribe/success');
           }
         }
       });
     }
     ```

   - On error: `toast.error(t('subscribe.checkoutError'))`.
   - Finally: clear loading state.

**SubscribeSuccessPage.jsx** — `/subscribe/success` confirmation page:

1. **Imports:** React, useState, useEffect, useNavigate, useTranslation, useUser, useSubscription, useQueryClient, Lucide icons (CheckCircle, Sparkles, Loader2, ArrowRight).

2. **State:** `const [timedOut, setTimedOut] = useState(false)`. Track whether we've exceeded the 10-second polling window.

3. **Subscription polling:** Use `useSubscription()` to get `isPremium` and `isLoading`.
   ```javascript
   const queryClient = useQueryClient();
   const { user } = useUser();

   useEffect(() => {
     if (isPremium || timedOut) return;

     let attempts = 0;
     const MAX_ATTEMPTS = 10;

     const interval = setInterval(() => {
       attempts++;
       queryClient.invalidateQueries({ queryKey: ['subscription', user?.id] });

       if (attempts >= MAX_ATTEMPTS) {
         clearInterval(interval);
         setTimedOut(true);
       }
     }, 1000);

     return () => clearInterval(interval);
   }, [isPremium, timedOut, user?.id, queryClient]);
   ```

4. **Confirmed state (isPremium === true within 10s):**
   - Celebratory layout with glass card.
   - Large CheckCircle icon: `text-green-400 w-16 h-16 mx-auto`
