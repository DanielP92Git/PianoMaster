## Implementation Notes for Executor

These notes capture the design boundary to prevent scope creep during execution:

1. **Do not change any component JSX** beyond removing local utility function definitions and adding import lines (QUAL-01, QUAL-02, QUAL-03).
2. **Do not modify any Tailwind classes** or CSS during this phase.
3. **Do not rename locale keys** — only change string values. Key renaming is deferred per CONTEXT.md D-07 and the Deferred Ideas section.
4. **TeacherDashboard lazy load** (QUAL-07): The UI renders identically before and after. Students never download the teacher bundle. Teachers see the same interface after the lazy chunk loads. No loading skeleton change is needed — the existing Suspense fallback at App.jsx:309 handles the loading state.
5. **AchievementsLegacy.jsx deletion** (QUAL-04): The live achievements page (`AchievementsRedesign`) renders identically after the dead file is removed. Only the stale comment at Achievements.jsx:2 needs removal alongside the file deletion.
