3. Generate and store `AUDIT_HMAC_SECRET`:


5. Deploy the Edge Function:
   ```bash
   supabase functions deploy process-account-deletions
   ```

6. Register the pg_cron schedule (run in SQL Editor):
```sql
SELECT cron.schedule(
  'process-account-deletions',
  '0 3 * * *',
  $$SELECT net.http_post(
    url := (SELECT decrypted_secret FROM vault.decrypted_secrets WHERE name = 'project_url') || '/functions/v1/process-account-deletions',
    headers := jsonb_build_object(
      'Content-Type', 'application/json',
      'x-cron-secret', (SELECT decrypted_secret FROM vault.decrypted_secrets WHERE name = 'cron_secret')
    ),
    body := '{}'::jsonb
  );$$
);
```

7. Verify the cron job was registered:
```sql
SELECT jobname, schedule FROM cron.job WHERE jobname = 'process-account-deletions';
```
Should return one row with schedule `0 3 * * *`.
