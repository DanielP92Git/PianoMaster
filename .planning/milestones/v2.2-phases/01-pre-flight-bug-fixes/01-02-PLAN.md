<objective>
Fix trail flag derivation and auto-grow boundary guard so accidental-aware trail sessions work correctly and natural-notes sessions are never contaminated with accidentals.

Purpose: Without this fix, trail-launched sessions for accidentals nodes always start with enableSharps/enableFlats=false (hardcoded), meaning no accidental answer options appear. Additionally, the auto-grow combo system can inject accidentals into natural-notes sessions, which would confuse learners who haven't learned sharps/flats yet.

Output: Fixed TrailNodeModal.jsx, fixed NotesRecognitionGame.jsx, fixed SightReadingGame.jsx, new auto-grow boundary tests.

    /**
     * Filter candidate notes from auto-grow based on accidental boundary rule.
     * Natural-only sessions cannot receive accidentals from next nodes.
     * Accidental sessions can receive any note (naturals or more accidentals).

    export function filterAutoGrowCandidates(candidatePool, currentPoolHasAccidentals) {

      return candidatePool.filter(pitch => !/[#b]/.test(pitch));
    }
