# Milestone v3.5: Rhythm Pedagogy

**Status:** ✅ SHIPPED 2026-06-29
**Phases:** 01
**Total Plans:** 10 (across 4 execution waves)

## Overview

Restructure the active rhythm trail into a pedagogically coherent **10-unit / 55-node** order
anchored by three falsifiable principles — **Pulse-first**, **Rests-woven**, **Concept-per-unit** —
and introduce 12 kid-friendly Duolingo-style intro/scaffolding screens (one per rhythmic concept)
that a child must acknowledge before exercises begin. Existing student rhythm progress is wiped on
deploy via an atomic Supabase migration; `students_score.total_xp` is preserved untouched. Rhythm
only — Treble / Bass / Ear-training trails are unchanged. The hidden Syncopation unit ("Off-Beat
Magic") stays hidden and was renamed `rhythm_8_*` → `rhythm_synco_*` to free the `rhythm_8_*`
numeric namespace for the new U8 (3/4 Meter) unit.
