```jsx

  aria-atomic="true"
  className="sr-only"

</div>
```
The `sr-only` class is a Tailwind utility that visually hides content but keeps it accessible to screen readers.

    animate={{ rotate: [0, 0, -90, -90, 0] }}

    {/* Piano keys overlay */}

    </div>

)}
