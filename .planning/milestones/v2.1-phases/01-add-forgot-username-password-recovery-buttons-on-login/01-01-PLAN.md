**In `src/services/apiAuth.js`**, add two new exported functions after the existing `logout` function:

1. `resetPassword({ email })` -- Calls `supabase.auth.resetPasswordForEmail(email, { redirectTo: siteUrl + '/reset-password' })`. Use the same `siteUrl` pattern already defined at the top of the file (derive from `process.env.NODE_ENV`). Note: The siteUrl variable is defined in supabase.js but NOT in apiAuth.js, so you need to compute it inline or import it. Compute it inline using the same pattern: `const isDevelopment = process.env.NODE_ENV === 'development'; const siteUrl = isDevelopment ? 'http://localhost:5174' : 'https://piano-master-nine.vercel.app';` -- place this at module level near the top of apiAuth.js (after imports). On error, throw with the error message. On success, return `{ data }`. Do NOT show a toast here -- the UI will handle messaging.

2. `updatePassword({ password })` -- Calls `supabase.auth.updateUser({ password })`. On error, throw with the error message. On success, return `{ data }`. No toast here either.

**Create `src/features/authentication/useResetPassword.js`**:
- Follow the `useLogin.js` pattern exactly.
- Import `useMutation` from `@tanstack/react-query`.
- Import `resetPassword as resetPasswordApi` from `../../services/apiAuth`.
- Import `useTranslation` from `react-i18next`.
- Export `useResetPassword()` hook returning `{ resetPassword, isPending, isSuccess, reset }`.
- `mutationFn`: calls `resetPasswordApi({ email })`.
- `onError`: Show toast with `t('auth.forgotPassword.errorGeneric')` as fallback. Do NOT show specific error details to avoid email enumeration -- always show a generic message.
- No `onSuccess` handler (the component manages success state via `isSuccess`).
- No retry (retry: 0) -- password reset should not auto-retry.
- Expose `reset` from useMutation so the component can reset isSuccess state.

**Create `src/features/authentication/useUpdatePassword.js`**:
- Follow the same pattern.
- Import `updatePassword as updatePasswordApi` from `../../services/apiAuth`.
- Import `useNavigate` from `react-router-dom` and `useTranslation`.
- Export `useUpdatePassword()` hook returning `{ updatePassword, isPending, isSuccess }`.
- `mutationFn`: calls `updatePasswordApi({ password })`.
- `onSuccess`: Show success toast with `t('auth.resetPassword.successMessage')`, then `navigate('/login', { replace: true })` after a 2-second delay so the user sees the success state.
- `onError`: Show toast with `t('auth.resetPassword.errorGeneric')`.
- No retry (retry: 0).
