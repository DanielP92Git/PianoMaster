## Critical Gaps (Blockers)

### BLOCKER-1: boss_rhythm_7 invisible on TrailMap

- **REQ:** PLAY-03
- **Where:** `src/data/units/rhythmUnit7Redesigned.js:377` — `isBoss: false` with comment "MINI_BOSS does NOT set isBoss: true"
- **Why:** `getNodesByCategory('rhythm')` filters category=='rhythm' (excludes), `getBossNodes()` filters isBoss===true (excludes). Falls through both → invisible.
- **Impact:** Unit 7 shows 5 nodes; Compound Commander unreachable.
- **Fix:** Flip to `isBoss: true`. NODE_TYPES.MINI_BOSS still controls overlay gating + timing tier.

### BLOCKER-2: PLAY-01 pulse hold path dormant

- **REQ:** PLAY-01 (pulse half)
- **Where:** `src/components/games/rhythm-games/MixedLessonGame.jsx:161-181` (`buildRhythmTapConfig` — no `beats` field) + `src/components/games/rhythm-games/renderers/PulseQuestion.jsx:115` (`config.beats || PULSE_BEATS`).
- **Why:** No trail node delivers long-duration pulse beats. PULSE_BEATS is 4 × `durationUnits:4` → `isHoldNote(4) === false` → hold mechanic never fires.
- **Fix:** (a) extend `buildRhythmTapConfig` to derive `beats` from `focusDurations`/`patternTags` and add a pulse exercise to a node with `durations: ['q','h']`, OR (b) narrow PLAY-01 to listen&tap-only and update REQUIREMENTS.md.
