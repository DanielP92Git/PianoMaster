**XPRing.jsx** — Circular SVG progress ring with gold star center icon.

Props interface:
```javascript
{ progressPercentage, xpCurrent, xpTotal, isMaxLevel, size = 100, reducedMotion = false }
```

Implementation:
- SVG with viewBox="0 0 120 120", two circles: background track (stroke white/20, strokeWidth 8) and progress arc (stroke gradient blue-to-indigo, strokeWidth 8)
- Progress arc uses stroke-dasharray and stroke-dashoffset for fill. Total circumference = 2 * PI * radius (radius ~48 to fit inside 120x120 with 8px stroke). Progress = circumference * (1 - progressPercentage/100). Apply `transform="rotate(-90 60 60)"` so arc starts at top
- If `!reducedMotion`, add CSS transition on stroke-dashoffset: `transition: stroke-dashoffset 1s ease-out`
- Center: render GoldStar component (import from `../ui/GoldStar`) with `size={28} filled glow`
- Below the ring (outside SVG, in the wrapper div): show `"{xpCurrent}/{xpTotal} XP"` in text-xs text-white/70. If isMaxLevel, show just `"{xpCurrent} XP"`
- Use SVG linearGradient defs for the progress stroke (from-blue-400 to-indigo-500 equivalent: #60A5FA to #6366F1)
- No useQuery inside this component — it receives data via props
