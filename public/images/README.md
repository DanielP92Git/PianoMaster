# Dashboard Hero Image

## Required Image

Please save the beautiful piano fantasy image as `dashboard-hero.jpg` in this directory.

### Image Requirements:
- **Filename:** `dashboard-hero.jpg` or `dashboard-hero.png`
- **Recommended Size:** 1920x800px (minimum)
- **Format:** JPG or PNG
- **Quality:** High resolution for best display on all devices

### Alternative Formats:
If you want to use a different image format, update the path in:
`src/components/layout/Dashboard.jsx` (line with `backgroundImage` property)

### Current Expected Path:
```
/images/dashboard-hero.jpg
```

This image will be displayed as the hero section at the top of the dashboard, with an elegant gradient overlay and smooth transition to the stats section below.
